## Numbered Lists
1.  Numbered list
2.  Item 2
    *   Nested list
    *   Another nested item
3.  Item 3

```markdown
1. list item 1
1. list item 2
    1. list item 2.1
    1. list item 2.2
    1. list item 2.3
1. list item 3
```
